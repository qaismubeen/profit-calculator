## Profit Calculator (C++ → Web)

This project calculates profit based on:
- Principal Amount
- Profit Rate
- 15% WHT Tax deductions (For Filer)
30% WHT deductions (For Non-Filer)

### Technologies
- C++ (original logic)
- HTML, CSS, JavaScript
- GitHub Pages
